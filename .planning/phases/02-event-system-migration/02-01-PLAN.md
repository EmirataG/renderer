1. In `src/types/verovio-augments.d.ts`, add `renderToTimemap()` to the VerovioToolkit class declaration. The method signature should be:
   ```
   renderToTimemap(options?: { includeMeasures?: boolean; includeRests?: boolean }): Array<{
     tstamp: number;
     qstamp: number;
     on?: string[];
     off?: string[];
     tempo?: number;
   }>;
   ```
   Do NOT pass `includeRests: true` anywhere -- the default (no rests) matches existing OSMD behavior which explicitly skips rests.

2. In `src/lib/getEvents.ts`, add a new exported function `getEventsFromVerovio()` alongside the existing `getEvents()` (do NOT remove the old function -- SyncEditor still uses OSMD).

   Signature:
   ```typescript
   export function getEventsFromVerovio(
     toolkit: VerovioToolkit,
     svgContainer: HTMLElement
   ): MusicalEventWithY[]
   ```

   Where `MusicalEventWithY` extends `MusicalEvent` with `{ y: number }`. Export this interface too.

   Implementation:
   a. Call `toolkit.renderToTimemap()` to get the full timemap array.
   b. Filter to entries where `entry.on` exists and `entry.on.length > 0` (note onset entries only).
   c. For each onset entry, create a MusicalEvent:
      - `id`: `"evt-${index}"` (sequential, matching existing convention)
      - `beatOnset`: `entry.qstamp / 4` (convert quarter-note units to whole-note fractions to match OSMD's RealValue convention that interpolation.ts expects)
      - `beatDuration`: 0 (calculated in next step)
      - `svgIds`: `entry.on` (Verovio note xml:id values -- these match SVG DOM id attributes directly, NO prefix needed)
      - `x`: 0 (not used for vertical camera scrolling)
   d. Calculate `beatDuration` for each event: `events[i].beatDuration = events[i+1].beatOnset - events[i].beatOnset`. Last event gets `beatDuration = 1` (matching existing convention in getEvents).
   e. Extract Y positions from DOM: for each event, use the first svgId to query the DOM:
      ```typescript
      const noteEl = svgContainer.querySelector(`#${CSS.escape(svgIds[0])}`);
      ```
      If found, compute Y relative to container:
      ```typescript
      const containerRect = svgContainer.getBoundingClientRect();
      const noteRect = noteEl.getBoundingClientRect();
      const y = noteRect.top - containerRect.top + noteRect.height / 2;
      ```
      If not found, y = 0.
   f. Return the MusicalEventWithY[] array.

   Import VerovioToolkit type: `import type { VerovioToolkit } from 'verovio/esm';`

   IMPORTANT: This function assumes toolkit.renderToMIDI() has already been called (Phase 1 ensures this in useVerovio.ts). Without renderToMIDI(), renderToTimemap() returns empty/zero results.
  </action>
