   - Replace `#000000` (case-insensitive) with `#111111`
   - Replace `rgb(0,0,0)` with `rgb(17,17,17)` (handles whitespace)
   - Use pre-compiled regex patterns at module scope for performance

2. **Core Conversion (TEX-01, TEX-02):**
   - `svgToTexture(svgString: string): Promise<Texture>`
   - Apply color preprocessing first
   - Create data URI: `data:image/svg+xml,${encodeURIComponent(processedSvg)}`
   - Create Image element, set src to data URI
   - Use `await image.decode()` (NOT onload callback)
   - Return `Texture.from(image)`

3. **Caching (TEX-03):**
   - Module-scope `textureCache = new Map<string, Texture>()`
   - `getCacheKey(svgString: string, scale: number, font: string): string`
     - Use content fingerprint: `${svgString.length}_${svgString.slice(0,100)}_${svgString.slice(-100)}_${scale}_${font}`
   - `getOrCreateTexture(svgString, scale, font): Promise<Texture>`
     - Check cache first, return if hit
     - Otherwise convert and cache result
   - `clearTextureCache(): void`
     - Iterate cache, call `texture.destroy(true)` for each

   - `getTextureCacheSize(): number` for debugging

     - Simply `await document.fonts.ready`
