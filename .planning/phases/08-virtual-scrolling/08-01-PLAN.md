1. Add `useMemo` for visible page indices calculation AFTER the useVerovio hook call:

```typescript
const visiblePageIndices = useMemo(() => {
  // Short scores: mount all pages
  if (pageCount <= 3) {
    return new Set(Array.from({ length: pageCount }, (_, i) => i));
  }

  // Render mode: mount all pages for Puppeteer frame capture
  if (isRenderMode) {
    return new Set(Array.from({ length: pageCount }, (_, i) => i));
  }

  // Find which page the camera Y position is on
  let currentPage = 0;
  for (let i = 0; i < pageOffsets.length; i++) {
    const pageEnd = pageOffsets[i + 1] ?? totalHeight;
    if (cameraY < pageEnd) {
      currentPage = i;
      break;
    }
  }

  // Build window: current page +/- 1
  const visible = new Set<number>();
  for (let i = Math.max(0, currentPage - 1); i <= Math.min(pageCount - 1, currentPage + 1); i++) {
    visible.add(i);
  }
  return visible;
}, [cameraY, pageOffsets, pageCount, totalHeight, isRenderMode]);
```

2. Update the page rendering JSX (around line 744) to conditionally render:

Replace:

```jsx
{svgPages.map((svg, i) => (
```

```jsx
{svgPages.map((svg, i) => (

      className="preview-score"

    />

```

Note: When a page is unmounted, we set its ref to null explicitly. This prevents stale refs and ensures computeEventPositions filters them out properly.
