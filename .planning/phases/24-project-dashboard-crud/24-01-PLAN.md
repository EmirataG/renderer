<task type="auto">
  <name>Task 2: Create project CRUD API Route Handlers</name>

  <action>
    1. Create helper function pattern (used in both route files):
       - Create a shared auth verification function at the top of each route file (or inline):
         ```
         async function getAuthenticatedUser() {
           const cookieStore = await cookies();
           const session = cookieStore.get('__session')?.value;
           if (!session) return null;
           try {
             return await adminAuth.verifySessionCookie(session, true);
           } catch {
             return null;
           }
         }
         ```
       - Import `cookies` from `next/headers`, `adminAuth` from `@/lib/firebase-admin`, `getDb` and `FieldValue` from `@/lib/firestore`.

    2. Create `src/app/api/projects/route.ts`:
       - **GET handler:** Verify auth (return 401 if not authenticated). Query Firestore: `db.collection('projects').where('userId', '==', user.uid).orderBy('updatedAt', 'desc').get()`. Map results: convert Firestore Timestamps to ISO strings using `.toDate().toISOString()`. Return `Response.json({ projects })`.
       - **POST handler:** Verify auth (return 401). Parse body: `{ name, viewMode }` from `request.json()`. Validate: `name` must be a non-empty string (return 400 if not). Generate ID: `crypto.randomUUID()`. Set document: `db.collection('projects').doc(projectId).set({ userId: user.uid, name, viewMode: viewMode || 'page', createdAt: FieldValue.serverTimestamp(), updatedAt: FieldValue.serverTimestamp() })`. Return `Response.json({ id: projectId }, { status: 201 })`.

    3. Create `src/app/api/projects/[id]/route.ts`:
       - **DELETE handler:** Verify auth (return 401). Extract `id` from params: `const { id } = await params`. Get the project document: `db.collection('projects').doc(id).get()`. If it doesn't exist, return 404. If `doc.data().userId !== user.uid`, return 403 (ownership check). Delete: `db.collection('projects').doc(id).delete()`. Return `Response.json({ status: 'deleted' })`.
       - The params type is `{ params: Promise<{ id: string }> }` (Next.js 16 async params).
  </action>

    - Route files import from `@/lib/firebase-admin` and `@/lib/firestore`
  </verify>
