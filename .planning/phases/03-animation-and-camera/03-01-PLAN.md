Fix the camera jitter bug during system-to-system transitions in sync playback, and verify transport controls and notehead animation are unaffected.

Purpose: The camera currently interpolates Y linearly between consecutive events even when they cross staff system boundaries (e.g., Y jumps from 120px to 450px), causing visible jittery up/down nudges. This fix detects system boundaries and snaps instead of interpolating.

Output: A working camera that does clean system-to-system snaps with CSS easing, while preserving smooth within-system tracking. All existing animation and transport behavior unchanged.
</objective>

  **Change 1: System boundary detection in animateSync() (lines 358-368)**

  Replace the Y interpolation block in `animateSync()` with system-boundary-aware logic:

  ```typescript
  // Calculate Y position — detect system jumps and snap instead of interpolating
  const SYSTEM_JUMP_THRESHOLD = 50; // pixels — system transitions have Y deltas > 50px
  const nextEvent = interpolatedEvents[index + 1];
  if (nextEvent && nextEvent.computedTimestamp > event.computedTimestamp) {
    const yDelta = Math.abs(nextEvent.y - event.y);
    if (yDelta > SYSTEM_JUMP_THRESHOLD) {
      // Cross-system: stay on current event's Y until we actually reach the next event
      currentYRef.current = event.y;
    } else {
      // Same system: smooth interpolation
      const progress =
        (currentTime - event.computedTimestamp) /
        (nextEvent.computedTimestamp - event.computedTimestamp);
      currentYRef.current =
        event.y + (nextEvent.y - event.y) * Math.min(1, progress);
    }
  } else {
    currentYRef.current = event.y;
  }
  ```

  **Change 2: Same fix in setTimestamp() (lines 514-528)**

  Apply the identical system-jump detection to the `setTimestamp` callback's Y interpolation block. Use the same `SYSTEM_JUMP_THRESHOLD` constant (extract it to module scope or duplicate inline — either is fine since it's one constant).

  ```typescript

  if (
    nextEvent &&
    nextEvent.computedTimestamp > currentEvent.computedTimestamp

    const yDelta = Math.abs(nextEvent.y - currentEvent.y);
    if (yDelta > SYSTEM_JUMP_THRESHOLD) {
      currentYRef.current = currentEvent.y;

        (seconds - currentEvent.computedTimestamp) /
  ```
