**Interface change:**

```typescript
interface UseVerovioResult {
  svgPages: string[];        // Individual page SVG strings (empty array when loading)
  pageHeights: number[];     // Actual pixel height of each page
  pageOffsets: number[];     // Cumulative Y offset for each page start
  totalHeight: number;       // Sum of all page heights
  pageCount: number;         // Number of pages
  toolkit: VerovioToolkit | null;
  isLoading: boolean;
  error: string | null;
}
```

**Option changes in setOptions():**
- Change `pageHeight: 60000` to `pageHeight: 2970` (Verovio A4 default -- enables pagination)
- Add `pageMarginTop: 0` (flush stacking -- removes default 50px margin)
- Add `pageMarginBottom: 0` (flush stacking)
- Keep all other existing options unchanged (svgViewBox, svgRemoveXlink, breaks, header, footer, scale, pageWidth)

**Rendering loop (replaces single renderToSVG(1)):**

```typescript
toolkit.loadData(xml);
toolkit.renderToMIDI();

const pageCount = toolkit.getPageCount();
const svgPages: string[] = [];
for (let i = 1; i <= pageCount; i++) {
  svgPages.push(toolkit.renderToSVG(i));
}
```

**Page height extraction (from SVG strings, no DOM):**

Parse the `height` attribute from each SVG string to build the offset table:

```typescript
function extractPageHeight(svgString: string): number {
  const match = svgString.match(/height="(\d+(?:\.\d+)?)px"/);
  if (match) return parseFloat(match[1]);
  // Fallback: parse viewBox
  const vbMatch = svgString.match(/viewBox="0 0 [\d.]+ ([\d.]+)"/);
  if (vbMatch) return parseFloat(vbMatch[1]);
  return 0;
}

const pageHeights = svgPages.map(extractPageHeight);
const pageOffsets: number[] = [];
let cumulative = 0;
for (const h of pageHeights) {
  pageOffsets.push(cumulative);
  cumulative += h;
}
const totalHeight = cumulative;
```
