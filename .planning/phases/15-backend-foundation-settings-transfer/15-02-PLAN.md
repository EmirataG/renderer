Create `src/routes/export.ts`:
- Import pipeline from 'node:stream/promises', createWriteStream from 'node:fs', writeFile from 'node:fs/promises', join and extname from 'node:path'
- Import { FastifyInstance } from 'fastify'
- Import { randomUUID } from 'node:crypto'
- Import jobManager, validation functions, createJobTempDir, cleanupTempDir, config
- Export default async function exportRoutes(fastify: FastifyInstance): void
- Register POST '/export' handler:
  1. const parts = request.parts()
  2. Initialize: settingsRaw: string | null = null, syncAnchorsRaw: string | null = null, uploadedFiles: Map<string, { path: string; mimetype: string }> = new Map()
  3. Generate jobId = randomUUID()
  4. Create tempDir = await createJobTempDir(jobId)
  5. try block: iterate parts:
     - type 'field': collect 'settings' and 'syncAnchors' as strings
     - type 'file': extract ext from part.filename via extname(), fall back to mimeToExt helper. For audio files, stream to disk using pipeline(part.file, createWriteStream(destPath)). For musicXml (typically <2MB), can use toBuffer() then writeFile. Store in uploadedFiles map.
  6. After iteration, validate required fields: settings, syncAnchors, musicXml file, audio file -- return 400 with specific message for each missing item
  7. JSON.parse settings and syncAnchors -- catch parse errors, return 400
  8. Call validateExportSettings(settings) -- if errors, return 400 with joined error messages
  9. Call validateSyncAnchors(syncAnchors) -- if errors, return 400 with joined error messages
  10. Create job via jobManager.createJob(jobId, tempDir, settings, syncAnchors)
  11. Return 201 { jobId: job.id, status: job.status }
  12. catch block: await cleanupTempDir(tempDir), then re-throw (Fastify error handler will format response)
- Helper function mimeToExt(mimetype: string): string -- maps common MIME types: audio/mpeg -> '.mp3', audio/wav -> '.wav', audio/ogg -> '.ogg', audio/mp4 -> '.m4a', application/xml -> '.xml', text/xml -> '.xml', image/png -> '.png', image/jpeg -> '.jpg', default -> '' (empty, let filename ext be used)

Create `src/routes/status.ts`:
- Export default async function statusRoutes(fastify: FastifyInstance): void
- Register GET '/export/:jobId/status' handler:
  1. Extract jobId from params
  2. const job = jobManager.getJob(jobId)
  3. If not found, return 404 { error: 'Job not found' }
  4. Return { jobId: job.id, status: job.status, createdAt: job.createdAt, completedAt: job.completedAt, error: job.error }
