## What Was Built

### 1. Zustand Store (unplayedStyleStore.ts)

Created a Zustand store following existing patterns (syncStore.ts, eventStore.ts):

```typescript
export type UnplayedStyleMode = 'dimmed' | 'invisible' | 'color';

interface UnplayedStyleStore {
  enabled: boolean;           // Toggle feature on/off
  mode: UnplayedStyleMode;    // Current styling mode
  dimOpacity: number;         // For 'dimmed' mode (0.3 default)
  unplayedColor: string;      // For 'color' mode (#666666 default)

  // Actions
  setEnabled, setMode, setDimOpacity, setUnplayedColor
}
```

### 2. Inspector UI Controls

Added "Unplayed Score Styling" section to Inspector panel:

- **Checkbox**: "Style Unplayed Score" toggle
- **Dropdown**: Mode selection (Dimmed, Invisible, Different Color)
- **Opacity slider**: Shown only for 'dimmed' mode (10-90%)
- **Color picker**: Shown only for 'color' mode

UI follows existing grunge styling patterns (grunge-checkbox, grunge-select, grunge-range, grunge-color-picker).
