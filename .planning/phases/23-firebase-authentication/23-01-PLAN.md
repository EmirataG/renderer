       - `POST` handler: Parse `{ idToken }` from request body. Call `adminAuth.createSessionCookie(idToken, { expiresIn: SESSION_EXPIRY_MS })`. Use `(await cookies()).set(SESSION_COOKIE_NAME, sessionCookie, { httpOnly: true, secure: process.env.NODE_ENV === 'production', sameSite: 'lax', path: '/', maxAge: SESSION_EXPIRY_MS / 1000 })`. Return `Response.json({ status: 'success' })`. Wrap in try/catch, return 401 on failure.
       - `DELETE` handler: `(await cookies()).delete(SESSION_COOKIE_NAME)`. Return `Response.json({ status: 'success' })`.

    2. Create `src/app/login/page.tsx` -- Server Component shell:
