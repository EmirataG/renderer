  <action>
**config.ts change:**
Change the port line from:
```typescript
port: 3001,
```
To:
```typescript
port: parseInt(process.env.PORT ?? '3001', 10),
```
This reads PORT from environment (Fly.io sets PORT=8080) with fallback to 3001 for local dev. Leave all other config values unchanged.

**server.ts change:**
After the `await server.listen(...)` call and before the closing brace of `main()`, add SIGTERM/SIGINT graceful shutdown:
```typescript
const shutdown = async () => {
  await server.close(); // Triggers onClose hooks (cleanup timer, browser pool)
  process.exit(0);
};
process.on('SIGTERM', shutdown);
process.on('SIGINT', shutdown);
```

This ensures Fly.io's auto-stop (which sends SIGTERM) triggers proper cleanup of the browser pool and cleanup timer before the process exits.
