  <action>
Create a new PixiJS-based renderer at `src/renderers/PixiSingleLineRenderer.tsx` following the @pixi/react v8 patterns from research.

**Component structure:**

1. **Module-level setup:**
   - Import Application, extend, useApplication from @pixi/react
   - Import Container, Sprite from pixi.js
   - Call extend({ Container, Sprite }) at module scope
   - Import useSingleLineVerovio hook
   - Import sectionsToTextures, clearTextureCache, TextureResult from svgToTexture

2. **Props interface:**
   ```typescript
   interface Props {
     xml: string;
     scoreScale?: number;  // Default 1
     musicFont?: string;   // Default 'Bravura'
   }
   ```

3. **ContextLossHandler child component:**
   - Uses useApplication() hook to get app instance
   - Registers webglcontextlost/webglcontextrestored event listeners on app.canvas
   - Logs to console: '[PixiRenderer] WebGL context lost' / 'restored'
   - Cleans up listeners in useEffect return
   - Renders null (invisible helper component)

4. **Main PixiSingleLineRenderer component:**
   - Call useSingleLineVerovio(xml, verovioScale, 15, musicFont) where verovioScale = Math.round(40 * scoreScale)
   - Use useState for textures: TextureResult[]
   - useEffect to convert sections to textures when sections change:
     - Guard: if sections.length === 0, return early
     - Set cancelled flag for cleanup
     - Call sectionsToTextures(sections, verovioScale, musicFont)
     - Set textures state if not cancelled
   - useEffect cleanup to call clearTextureCache() on unmount
   - Loading state: if isLoading || textures.length === 0, return loading div

5. **Render structure:**
   ```tsx
   <Application
     width={totalWidth}
     height={maxHeight}
     backgroundColor={0xffffff}
     backgroundAlpha={0}
   >
     <ContextLossHandler />
     <pixiContainer x={0} y={0}>
       {textures.map((result, index) => (
         <pixiSprite
           key={index}
           texture={result.texture}
           x={sectionOffsets[index]}
           y={0}
         />
       ))}
     </pixiContainer>
   </Application>
   ```

**Key patterns (from 15-RESEARCH.md):**
- Use lowercase pixi-prefixed JSX: `<pixiSprite>`, `<pixiContainer>`
- extend() must be called at module level, not in component body
- useApplication() only works in CHILD components, not same component as Application
- Stage dimensions from hook: totalWidth, maxHeight
- Sprite positions from hook: sectionOffsets[index] for X, 0 for Y

**Requirements covered:**

- REN-03: Stage dimensions match totalWidth/maxHeight
- REN-04: clearTextureCache() called in useEffect cleanup
- REN-05: Context loss handlers registered in ContextLossHandler
