**Create `src/lib/autoSave.ts`:**

This module subscribes to projectStore and syncStore OUTSIDE React and triggers debounced PATCH saves. It returns an unsubscribe function for cleanup.

```typescript
import { useProjectStore } from '../stores/projectStore';
import { useSyncStore } from '../stores/syncStore';

const SAVE_DEBOUNCE_MS = 1500;
const SAVED_DISPLAY_MS = 3000;
```

**`getSaveableSettings` helper:** Extract only settings keys from the project store state (exclude projectId, saveStatus, lastSaveError, and all action functions). Use destructuring to pull out non-settings fields, spread the rest.

**`performSave(projectId: string)` async function:**
1. Set saveStatus to `'saving'` via `useProjectStore.getState().setSaveStatus('saving')`
2. Read current settings via `getSaveableSettings(useProjectStore.getState())`
3. Read current anchors via `useSyncStore.getState().anchors`
4. Serialize anchors: `Object.fromEntries(anchors)`
5. `fetch(\`/api/projects/${projectId}\`, { method: 'PATCH', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ settings, anchors: serializedAnchors }) })`
6. If response ok: set saveStatus to `'saved'`, then `setTimeout(() => setSaveStatus('idle'), SAVED_DISPLAY_MS)` to auto-dismiss after 3 seconds
7. If response not ok: set saveStatus to `'error'` with response statusText
8. Catch block: set saveStatus to `'error'` with error message

**`scheduleSave()` function:**
1. Read `projectId` from `useProjectStore.getState().projectId`
2. If no projectId, return early (no project to save to)
3. Clear any existing `saveTimer` via `clearTimeout`
4. Set new `saveTimer = setTimeout(() => performSave(projectId), SAVE_DEBOUNCE_MS)`

**`initAutoSave()` exported function:**
Returns an unsubscribe function (call to tear down subscriptions + clear timer).

Subscribe to project settings changes:
```typescript
const unsub1 = useProjectStore.subscribe(
  (state) => getSaveableSettings(state),
  () => scheduleSave(),
  { equalityFn: (a, b) => JSON.stringify(a) === JSON.stringify(b) }
);
```

Subscribe to sync anchor changes:
```typescript
const unsub2 = useSyncStore.subscribe(
  (state) => state.anchors,
  () => scheduleSave(),
  {
    equalityFn: (a, b) => {
      if (a.size !== b.size) return false;
      for (const [k, v] of a) {
        if (b.get(k) !== v) return false;
      }
      return true;
    }
  }
);
```

The Map equality function avoids JSON.stringify on every Map change (which would be expensive). It does a direct key/value comparison.

Return cleanup function:
```typescript
return () => {
  unsub1();
  unsub2();
  if (saveTimer) clearTimeout(saveTimer);
};
```

CRITICAL: The `savedDismissTimer` (the 3-second auto-dismiss) should also be cleared in the cleanup. Track it as a module-level variable like `saveTimer`.

**Create `src/components/SaveIndicator.tsx`:**

A small component that reads `saveStatus` from projectStore and renders status text.

```tsx
import { useProjectStore } from '../stores/projectStore';

export function SaveIndicator() {
  const saveStatus = useProjectStore((s) => s.saveStatus);
  const lastSaveError = useProjectStore((s) => s.lastSaveError);

  if (saveStatus === 'idle') return null;

  return (
    <div className="flex items-center gap-1.5 text-xs">
      {saveStatus === 'saving' && (
        <span className="text-neutral-400">Saving...</span>
      )}
      {saveStatus === 'saved' && (
        <span className="text-green-500">Saved</span>
      )}
      {saveStatus === 'error' && (
        <span className="text-red-400" title={lastSaveError ?? undefined}>Save error</span>
      )}
    </div>
  );
}
```

Keep it minimal. No animations or icons. The error state shows a title attribute with the error message on hover for debugging.

Store the cleanup function and call it on unmount or when projectId changes. Use a ref:

```typescript
const autoSaveCleanupRef = useRef<(() => void) | null>(null);
```

In the loadProject useEffect:
```typescript
useEffect(() => {
  // Clean up previous auto-save subscription if projectId changes
  autoSaveCleanupRef.current?.();
  autoSaveCleanupRef.current = null;

  if (!projectId) return;

  async function loadProject() {
    // ... existing load code ...
```
